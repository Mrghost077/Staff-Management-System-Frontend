# TeachGrid - School Staff Management System (Frontend) 🖥️

Frontend application for **TeachGrid**, a full-stack staff management system designed to streamline school operations such as attendance tracking, timetable management, and staff coordination.

---

## 🛠 Tech Stack

* **Framework:** React.js
* **Styling:** Tailwind CSS
* **State Management:** React Hooks & Context API
* **API Communication:** Axios
* **Routing:** React Router
* **Design:** Figma (High-fidelity wireframes)
* **Deployment:** Netlify

---

## ✨ Core Features

* 🔐 **Authentication System**
  Secure login and registration integrated with backend using JWT stored in HTTP-only cookies.

* 📊 **Interactive Dashboard**
  Displays real-time insights such as attendance, schedules, and announcements.

* 👨‍🏫 **Role-Based Access**
  Separate interfaces and functionalities for **Admin** and **Teacher** users.

* 📅 **Timetable & Attendance Management**
  Allows teachers and administrators to manage schedules and track attendance efficiently.

* 📢 **Announcements Module**
  Enables communication between administration and staff.

* 📱 **Responsive UI**
  Optimized for desktop and tablet devices for better usability.

---

## 🔗 Backend Integration

The frontend communicates with the backend via RESTful APIs.

* Uses **Axios** for API requests
* Sends and receives **JSON data**
* Authentication handled via **cookies (JWT-based)**
* Protected routes ensure only authorized users can access certain pages
